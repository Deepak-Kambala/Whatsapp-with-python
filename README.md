# WhatsApp Automation with Python

This is a simple Python project that allows you to send WhatsApp messages automatically at a scheduled time using the `pywhatkit` library.

---

## Features

- Send a WhatsApp message to any number with a country code.
- Schedule messages at a specific time (24-hour format).
- Optionally, close the browser tab automatically after sending the message.

---

## Requirements

- Python 3.x
- Google Chrome installed (or your default web browser)
- `pywhatkit` Python library
